# 大和配列（Yamato Layout）サイト構成案

参考: [大西配列](https://o24.works/layout/)
URL: `honjoh.dev/layout`

## 1. 構成案（ドラフト）

### A. ヒーロー (Hero)
- **キャッチコピー案**: 「日本語本来の並びを、指先に。」「30年の常識を、1ヶ月で超える。」
- **概要**: 50音順配列 × 清濁分離。直感的に覚えられ、疲れにくい「大和配列」。

### B. 特徴 (Features)
- **習得の容易性**: 「あかさたな...」の並びがベース。直感的に配置がわかる。
- **疲労軽減**: 手首・肘を動かさず入力可能。寝転がりながらでも打てる。（省スペース設計）
- **高速入力**: 開発1ヶ月半でe-typing「Fast」スコア(300)到達の実績。
- **QWERTY互換機能**: `Ctrl`, `Alt`, `Win` キー押下時はQWERTYに戻る。ショートカット(Ctrl+C等)がそのまま使える（**実用上の大きな強み**）。

### C. 導入方法 (Installation)
- **Windows**: AutoHotKey (スクリプト配布予定)
    - 記事内のスクリプトを `yamato.ahk` としてダウンロード可能にする。
- **macOS**: (今回は対象外、または「募集中」とする)
- **Linux/自作キーボード**: (Remap等のリンク紹介)

### D. コンテンツ構成案
- **Intro**: 開発経緯（QWERTYへの疑問 vs 1.5ヶ月の成果）
- **Layout**: 配列図（画像生成 or CSSグリッドで表現）
